feat: Add multi-currency support for recurring invoices

This commit implements comprehensive currency support for recurring invoices
and templates, completing the multi-currency feature across the application.

## Recurring Invoice Currency Features
- Added currency field to RecurringInvoiceTemplate model in database schema
- Currency selection in recurring invoice form with organization default fallback
- Currency display in recurring invoice table (amount column with currency code)
- Currency display in recurring invoice detail view and overview pages
- Currency support in invoice generation (cron and manual) with template currency
- All currency formatting uses template currency or organization default

## Database Changes
- Added currency column to recurring_invoice_templates table
- Migration: 20260126210906_add_currency_to_recurring_invoices
- Currency field is optional, falls back to organization defaultCurrency

## UI/UX Improvements
- Currency dropdown in recurring invoice form with 20+ supported currencies
- Currency code displayed below amounts in table for clarity
- Currency field shown in template details section
- All monetary values formatted with correct currency throughout

## API Updates
- POST /api/recurring-invoices now accepts currency parameter
- PUT /api/recurring-invoices/[id] supports currency updates
- Invoice generation routes use template currency when creating invoices
- Automatic fallback to organization defaultCurrency if template currency not set

## TypeScript Updates
- Updated RecurringInvoiceTemplate interface to include currency field
- Updated mutation hooks to support currency parameter
- Added proper type safety for currency handling

## Files Changed
- prisma/schema.prisma - Added currency field to RecurringInvoiceTemplate
- prisma/migrations/ - Added currency migration
- src/features/invoicing/components/recurring-invoice-form.tsx - Currency selection
- src/features/invoicing/components/recurring-invoice-tables/columns.tsx - Currency display
- src/features/invoicing/components/recurring-invoice-view.tsx - Currency in detail view
- src/app/dashboard/recurring-invoices/[id]/overview/page.tsx - Currency in overview
- src/app/api/recurring-invoices/route.ts - Currency handling in API
- src/app/api/recurring-invoices/[id]/route.ts - Currency updates
- src/app/api/recurring-invoices/[id]/generate/route.ts - Currency in generation
- src/app/api/cron/generate-recurring/route.ts - Currency in cron generation
- src/features/invoicing/hooks/use-recurring-invoices.ts - TypeScript types
- README.md - Updated roadmap to mark multi-currency support as completed

This completes the multi-currency support feature, allowing users to set and
display currencies for both regular invoices and recurring invoice templates
with proper fallback handling and consistent formatting throughout the app.
